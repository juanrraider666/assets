Example Carousel Page

Slides have variable heights. Transitions are smooth vertically as well horizontally. Bootstrap only provides for smooth transitions side-to-side. This custom script provides similar behavior up-and-down.

Accessibility in carousels is not great. This example page does not include all available accessibility features and exists only to demonstrate height change functionality. Users of this code should carefully consider accessibility needs and adapt the code appropriately.
